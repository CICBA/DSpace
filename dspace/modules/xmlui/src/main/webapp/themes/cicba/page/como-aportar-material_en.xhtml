HOW TO CONTRIBUTE MATERIAL?

If you are a researcher, a scholar or a member from the support staff from CIC, or if you have been so in the past, you can contribute with your work to the CIC-DIGITAL repository by following a simple procedure called self-archive1. It can be done from any place with Internet access and it allows you to send digitalized material. In order to do it, these steps must be followed:

  1. Login to the site (if you do not have a user account yet, you can create one here.)
  2. Go to "Upload material". Once there, the system will guide you through a number of simple steps.
  3. Complete a form with information of the work to be uploaded.
  4. Select the appropriate license of use.

Once CIC-DIGITAL receives the material, it will be published on the data bank after a brief verification, and you will be properly notified.
